Get Started

Where to apply transition?

A transition should be applied to any element that has a change of state. Adding a transition makes the change smoother instead of having a harsh quick change. Some examples:

  • A link that changes color on hover/focus
  • A menu that opens
  • An element that fades in/out like a pop-up

Where to put the transition rule?

When applying the actual CSS code, it is important to put the transition rule on the initial css rule set for the element. This allows for a seamless transition in and out of the changed state. If you apply the transition on the changed state (for example on the :hover), it will only apply when that state is applied, causing inconsistent transitions in and out of the state.
